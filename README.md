# How to install

Requirements:
- [Node.js](https://nodejs.org/en) (>=20.9.0)
- [npm](https://www.npmjs.com/) (>=10.2.1)
- [Git](https://git-scm.com/) (>=2.33.0)
- [Yarn](https://yarnpkg.com/) (>=1.22.19)

Clone the repository:
```sh
git clone https://github.com/minotaa/Whisper
```

Focus into the directory:
```sh
cd Whisper
```

Install dependencies:
```sh
npx yarn install 
```

Build project:
```sh
npx tsc
```

Run:
```sh
node .
```
